# Lens Project Overview

## Purpose

Lens is maintained as a developer workflow UI. The project work is organized around repeatable setup, visible operational notes, and small reviewable changes.

## Current Focus

- Keep the default branch runnable from a fresh clone.
- Document the expected local workflow before changing implementation details.
- Capture examples and validation steps next to the code they exercise.
- Track release notes so changes can be reviewed without reading every diff.

## Technology Area

Primary scope: TypeScript, Vite, and server integration.
